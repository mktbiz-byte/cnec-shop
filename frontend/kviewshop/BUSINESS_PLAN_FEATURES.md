# KviewShop - 사업계획서용 전체 기능 명세

## 1. 서비스 개요

**KviewShop**은 K-Beauty 크리에이터 커머스 플랫폼으로, 크리에이터가 자신만의 독립적인 온라인 몰을 개설하고, 기업이 등록한 상품을 큐레이션하여 판매하는 소셜 커머스 모델입니다.

### 비전
"모든 크리에이터가 자신만의 브랜드 몰을 갖는 세상"

### 핵심 가치
- 크리에이터 이코노미 활성화
- K-Beauty의 글로벌 유통 혁신
- 구매자-크리에이터-기업 삼자 Win-Win 구조

---

## 2. 회원 유형 (4가지)

### 2.1 통합 관리자 (Super Admin)
- **역할**: 플랫폼 전체 운영 관리
- **주요 기능**:
  - 기업 가입 승인/관리
  - 크리에이터 승인 및 레벨 관리
  - 정산 관리 (크리에이터, 기업별 월정산)
  - 매출 통계 대시보드 (국가별, 기간별)
  - MoCRA 미국 화장품 규제 모니터링
  - 법적 콘텐츠(약관, 개인정보방침) 관리
  - 리뷰/게시판 신고 관리

### 2.2 기업 관리자 (Brand Admin)
- **역할**: 상품 공급 기업 (브랜드, 제조사)
- **주요 기능**:
  - 회사 정보 등록 (한/영/일 다국어)
  - 상품 등록 및 관리 (다국어, 다통화)
  - 배송 가능 국가 설정 (JP, US, KR 등 15개국)
  - 인증서 관리 (KFDA, ISO22716, FDA, Halal, Vegan 등)
  - 크리에이터 수수료율 설정 (5~50%, 단계별 수수료)
  - 주문 관리 및 배송 추적
  - 월간 정산 확인
  - 샘플 박스 요청 관리 (크리에이터에게 체험 상품 발송)
- **의도**: 기업이 자체 유통 채널 없이도 크리에이터 네트워크를 통해 글로벌 시장에 진출

### 2.3 크리에이터 (Creator/Seller)
- **역할**: 콘텐츠 크리에이터, 인플루언서, 셀러
- **주요 기능**:
  - 개인 몰(셀러 페이지) 구성 및 커스터마이징
  - 기업 상품 큐레이션 (Pick) 및 순서 설정
  - 단축 URL 생성 (SNS 프로모션용)
  - 커뮤니티 게시판/채팅 설정
  - 라이브 쇼핑 세션 관리
  - 라이브 도우미 봇 설정
  - 주문 현황 확인
  - 수익/정산 관리
  - 레벨 시스템 (Bronze → Diamond)
- **의도**: 크리에이터가 상품 재고/배송 부담 없이 자신의 영향력으로 수익 창출

### 2.4 구매자 (Buyer)
- **역할**: 일반 소비자, 크리에이터 몰 이용자
- **주요 기능**:
  - 크리에이터 몰 구독 및 알림
  - 상품 구매 및 주문 추적
  - 장바구니 및 위시리스트
  - 상품 리뷰 작성 (적립금 제공)
  - Q&A 질문 등록
  - 커뮤니티 참여
  - 적립금 관리
  - 크리에이터 전환 신청
- **의도**: 구매자가 단순 소비자를 넘어 충성 팬이 되고, 최종적으로 크리에이터로 전환되는 선순환 구조

## 3. 핵심 기능 상세

### 3.1 단축 URL 시스템

**기능 설명**:
크리에이터가 자신의 몰에 대한 단축 URL을 생성하여 인스타그램, 유튜브, 틱톡 등 SNS에 쉽게 공유할 수 있는 시스템.

**세부 사항**:
- URL 형식: `kviewshop.com/s/{코드}` (3~20자, 영소문자/숫자/밑줄)
- 플랫폼별 전용 링크 생성 (Instagram용, YouTube용, TikTok용)
- 클릭 수, 접속 국가, 디바이스 타입 등 분석 데이터 제공
- Primary 링크 지정 기능
- 만료일 설정 가능

**의도**:
인스타그램 프로필, 유튜브 설명란 등에서 긴 URL 대신 깔끔한 단축 URL 사용. 각 플랫폼별 유입 경로를 분석하여 마케팅 효과 측정.

**수익 모델 연계**: 프리미엄 크리에이터에게 커스텀 도메인 연결 옵션 제공.

---

### 3.2 크리에이터 몰 커스터마이징

**기능 설명**:
크리에이터가 자신만의 독창적인 온라인 몰을 꾸밀 수 있는 에디터. 왼쪽 편집 패널, 오른쪽 실시간 미리보기 구조.

**세부 사항**:
- 프로필: 표시명, 바이오, 프로필 이미지
- 테마 색상: 9가지 액센트 컬러 (Gold, Rose, Blue, Purple 등)
- 배경 색상: Black/White 및 커스텀 (Navy, Forest, Wine 등)
- 소셜 미디어 링크 (Instagram, YouTube, TikTok)
- 공지사항 배너 (활성/비활성 전환)
- 커뮤니티 활성화 여부 설정
- 푸터 표시 설정
- 구독자 수 표시 여부
- 레이아웃 설정 (Grid/List)

**의도**:
크리에이터 개성이 담긴 몰로 팬 충성도 향상. 전문 쇼핑몰 제작 비용 없이 누구나 자신만의 브랜드 몰 운영 가능.

---

### 3.3 구매자 로그인 & 대시보드

**기능 설명**:
메인 화면이 구매자 로그인으로 시작. 로그인 후 대시보드에서 구독한 몰, 주문 내역, 적립금 등을 확인.

**대시보드 구성**:
- 통계 카드: 총 주문, 적립금 잔액, 구독 몰 수, 리뷰 수
- 구독 몰 목록 (빠른 접근)
- 최근 주문 내역
- 적립금 현황 및 리뷰 작성 유도 배너

**의도**:
구매자 경험 최적화. 크리에이터 몰 발견과 구독을 통해 재방문율 향상. 적립금과 리뷰 시스템으로 참여 유도.

---

### 3.4 구독 서비스

**기능 설명**:
구매자가 좋아하는 크리에이터 몰을 구독하여 새 상품, 세일, 라이브 방송 알림을 받는 시스템.

**세부 사항**:
- 몰 구독/구독취소
- 알림 설정: 신상품, 세일, 라이브 방송 각각 ON/OFF
- 구독한 몰 목록 (크리에이터 프로필, 테마 컬러 표시)
- 크리에이터 측에서 구독자 수 확인 가능

**의도**:
구매자-크리에이터 간 지속적 관계 구축. 푸시 알림을 통한 재방문율과 재구매율 향상. 크리에이터에게는 안정적인 팬베이스 형성.

---

### 3.5 법적 준수 푸터 (한국/미국/일본)

**기능 설명**:
각 몰 하단에 국가별 법적 요구사항을 충족하는 풋 페이지 자동 생성.

**한국 (KR)**:
- 이용약관, 개인정보처리방침
- 사업자정보 (상호, 대표, 사업자등록번호, 통신판매업신고)
- 환불정책 (전자상거래법 준수)
- 공정거래위원회 표준약관

**미국 (US)**:
- Terms of Service, Privacy Policy
- Business Information
- Refund Policy (30-day policy)
- FTC Act 준수

**일본 (JP)**:
- 利用規約, プライバシーポリシー
- 特定商取引法に基づく表記
- 返品・返金ポリシー

**의도**:
카드사 심사(PG 심사) 통과에 필수. 각 국가별 전자상거래법/소비자보호법 준수. 법적 리스크 최소화 및 신뢰도 확보.

---

### 3.6 커뮤니티 시스템 (게시판/채팅)

**기능 설명**:
각 크리에이터 몰에 구매자들이 자유롭게 소통할 수 있는 공간. 크리에이터가 게시판 또는 채팅 중 선택.

**세부 사항**:
- 게시판 모드: 글 작성, 댓글, 대댓글, 좋아요, 고정 글
- 글 유형: 일반, 리뷰, 질문, 공지
- 크리에이터 모더레이션 (글 숨김, 고정)
- 이미지 첨부 가능
- 조회수 / 좋아요 / 댓글 수 표시

**의도**:
몰 내 커뮤니티 형성으로 구매자 체류 시간 증가. 구매자끼리 상품 후기와 추천을 공유하여 전환율 향상. 크리에이터 팬덤 강화.

---

### 3.7 상품 Q&A 및 리뷰 시스템 (적립금 연동)

**기능 설명**:
각 상품에 Q&A와 리뷰를 등록할 수 있으며, 리뷰 작성 시 적립금이 지급되는 시스템.

**적립금 정책**:
- 텍스트 리뷰: **500원 (500P)**
- 인스타그램 게시물 연동 리뷰: **1,000원 (1,000P)**
- 적립금은 다음 구매 시 사용 가능

**Q&A 시스템**:
- 구매자가 상품에 질문 등록
- 브랜드 관리자 또는 크리에이터가 답변
- 공개/비공개 설정
- 답변 완료 상태 표시

**리뷰 시스템**:
- 별점 (1~5점)
- 텍스트 + 이미지 첨부
- 인스타그램 게시물 URL 연동
- "도움이 됐어요" 투표
- 인증구매 배지
- 베스트 리뷰 선정

**의도**:
인스타그램 연동 리뷰로 자연스러운 바이럴 마케팅 효과. 적립금으로 리뷰 작성 동기 부여. UGC(사용자 생성 콘텐츠) 확보로 신규 구매자 전환율 향상.

---

### 3.8 크리에이터 레벨 시스템 (게이미피케이션)

**기능 설명**:
판매 실적에 따른 등급 부여로 지속적 동기 부여.

**등급 체계**:

| 등급 | 필요 포인트 | 추가 커미션 | 색상 | 혜택 |
|------|-----------|-----------|------|------|
| **Bronze** | 0 | +0% | #CD7F32 | 기본 커미션율, 월간 리포트 |
| **Silver** | 1,000 | +1% | #C0C0C0 | 우선 샘플 배송, 월간 리포트 |
| **Gold** | 5,000 | +2% | #FFD700 | 전용 담당자 배정, 프로모션 우선 참여 |
| **Platinum** | 15,000 | +3% | #E5E4E2 | 독점 상품 액세스, 마케팅 지원 |
| **Diamond** | 50,000 | +5% | #B9F2FF | 브랜드 콜라보 기회, VIP 이벤트 초대, 전용 마케팅 예산 |

**세부 사항**:
- 판매 완료 주문마다 포인트 적립
- 레벨 업 시 알림 및 히스토리 기록
- 배지 시스템 (몰에 등급 배지 표시)
- 월간 랭킹 제공

**의도**:
게이미피케이션으로 크리에이터의 판매 동기 지속적 강화. 상위 등급 혜택으로 이탈 방지. 등급 배지가 몰 신뢰도를 높여 구매 전환율 향상.

---

### 3.9 라이브 쇼핑 연동

**기능 설명**:
인스타그램/유튜브/틱톡 라이브 중 실시간 구매 링크를 제공하여 FOMO 심리를 활용한 전환율 극대화.

**세부 사항**:
- 라이브 세션 스케줄링 (제목, 설명, 플랫폼, 시간)
- 플랫폼 연동: Instagram Live, YouTube Live, TikTok Live
- Featured 상품 지정 (라이브 전용 가격 설정 가능)
- 실시간 통계: 최대 시청자 수, 총 시청자, 주문 수, 매출
- 라이브 중 상품 링크 자동 게시

**의도**:
라이브 쇼핑은 전환율이 일반 커머스 대비 3~10배 높음 (McKinsey 보고). 크리에이터의 실시간 소통이 구매 결정을 가속화. FOMO(Fear of Missing Out) 효과로 즉각적 구매 유도.

---

### 3.10 라이브 도우미 기능 (Auto-Chat Bot)

**기능 설명**:
자동 채팅 시스템으로 라이브 방송 중 봇이 구매 링크를 시간에 맞춰 올려주는 방식.

**세부 사항**:
- 환영 메시지 자동 발송 (라이브 시작 시)
- 상품 링크 자동 게시 (설정된 간격, 기본 5분)
- 키워드 자동 응답 (예: "가격" → "상품 가격은 화면에서 확인해주세요!")
- 스케줄 메시지 설정 (시작 후 N초에 특정 메시지 발송)
- 봇 ON/OFF 토글 (크리에이터가 라이브마다 결정)
- 채팅 메시지 핀 고정 기능

**키워드 응답 예시**:
```
"가격" → "상품 가격은 화면에서 확인해주세요! 라이브 특가 진행 중!"
"배송" → "배송은 2-3일 소요됩니다. 해외 배송은 7-14일!"
"환불" → "상품 수령 후 7일 이내 환불 가능합니다."
"쿠폰" → "지금 구매하시면 다음 구매 시 10% 할인 쿠폰을 드립니다!"
```

**의도**:
크리에이터가 라이브 중 상품 설명에 집중할 수 있도록 반복 질문은 봇이 처리. 구매 링크 자동 게시로 전환 기회 극대화. 채팅이 활발해 보이는 효과로 신규 시청자 이탈 방지.

---

### 3.11 구매자 → 크리에이터 전환 시스템

**기능 설명**:
충성 고객을 크리에이터로 자동 전환하여 크리에이터 풀을 자동 확장.

**전환 조건 (기본값)**:
- 최소 5회 이상 주문
- 최소 3개 이상 리뷰 작성
- 총 $100 이상 구매
- 계정 생성 후 30일 이상

**전환 프로세스**:
1. 조건 충족 시 대시보드에 "Become a Creator" 버튼 활성화
2. 신청서 작성 (희망 유저네임, SNS 계정, 콘텐츠 계획 등)
3. 관리자 검토 및 승인
4. 크리에이터 계정 자동 생성
5. 기존 구매 이력 유지

**진행 상황 표시**:
- 각 조건별 달성률 프로그레스 바
- 조건 충족/미충족 체크 표시

**의도**:
구매자→크리에이터 전환으로 플랫폼 내 자연적인 셀러 확대. 이미 상품을 잘 아는 고객이 가장 좋은 셀러. 별도 크리에이터 모집 비용 절감. 충성 고객의 플랫폼 lock-in 효과.

## 7. 차별점

### 경쟁사 대비 차별화
1. **크리에이터 독립 몰**: 일반 마켓플레이스와 달리 크리에이터만의 독립적인 브랜드 몰
2. **단축 URL 시스템**: SNS 프로모션에 최적화된 URL 관리 및 분석
3. **4단계 회원 생태계**: 구매자→크리에이터 전환으로 자연 성장하는 생태계
4. **라이브 커머스 통합**: SNS 라이브와 연동된 실시간 구매 시스템
5. **AI 챗봇 도우미**: 라이브 중 자동 상품 링크 게시 및 FAQ 응답
6. **게이미피케이션**: 크리에이터 레벨 시스템으로 지속적 동기 부여
7. **글로벌 첫날부터**: 11개 언어, 다통화, 3개국 법적 준수
8. **인스타 리뷰 적립금**: 바이럴 마케팅과 리뷰를 동시에 달성
